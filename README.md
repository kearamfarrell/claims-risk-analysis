# Healthcare Claims Cost Analysis

## Overview
This project is an end-to-end analysis of healthcare insurance claims using Excel, SQL, and Power BI. The goal is to identify key cost drivers and risk factors affecting insurance charges.

## Tools Used
- Excel (data preparation)
- SQL for data analysis
- Power BI for visualization

## Key Analyses

### 1. Smoking Impact
Analyzed the relationship between smoking status and healthcare costs. Found that smokers consistently incur significantly higher average charges than non-smokers.

### 2. Regional Variation
Examined cost differences across geographic regions and found only minor variation compared to behavioral risk factors.

### 3. Risk Factors (BMI + Smoking)
Segmented patients by BMI categories and smoking status to identify combined effects on healthcare costs.

### 4. High-Cost Outliers
Used percentile-based filtering to identify the top 5% of highest-cost claims.

## Key Insight
Smoking status is the strongest predictor of healthcare costs, outweighing regional and BMI-related variation.
